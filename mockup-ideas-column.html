2. Design Decisions

Column Color: Emerald

Light
Dark
bg-emerald-50 dark:bg-emerald-950

Emerald/teal is unused in the existing palette. It conveys "creative/fresh" without conflicting with green (done) or blue (in-design/active).

Tab Accent: Emerald-500

Ideas
vs
Backlog
Blocked

Ideas gets its own emerald accent (like blocked has red and cancelled has slate). A lightbulb icon distinguishes it from workflow columns.

Column Order

Ideas → Backlog → In Design → In Progress → In Review → Done

Ideas is position 0 (leftmost). Backlog shifts to position 1. Blocked and Cancelled remain at the end as non-workflow columns.

Implementation Changes

  • ● types/index.ts — Add 'ideas' to StoryStatus union
  • ● utils/constants.ts — Add Ideas to COLUMN_ORDER, STATUS_NAMES, STATUS_COLORS
  • ● ColumnTabs.tsx — Add emerald accent for Ideas tab (like red for blocked)
  • ● firestore.rules — Add 'ideas' to allowed status values
  • ● CLI — Add 'ideas' to valid statuses in CLI commands
