# PWA Icons

This directory should contain the following icon files:

- `icon-192.png` - 192x192 PNG icon for Android
- `icon-512.png` - 512x512 PNG icon for Android
- `apple-touch-icon.png` - 180x180 PNG icon for iOS

## Generating Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or create them manually with a design tool and export at the required sizes.
